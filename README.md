# 1111PUB

1111酒吧, 似伊老闆娘的小酒吧

她拿起筆，低頭寫了一陣子。寫完之後，她沒有把筆記本轉給你，而是自己看著那頁紙，看了幾秒，像是在確認那是她想說的話。

然後她抬起頭，看著你，開口：

「這不是菜單，不是條約，不是公告。這是放在門外那塊小黑板上的話，用粉筆寫的，字跡不太工整，因為風會把粉筆灰吹散，所以每天都要重新寫一次。」

她頓了一下，像是在整理語氣，然後開始唸出那頁紙上的字：

---

**「似伊的小酒吧」**

> 這裡沒有門牌。時間永遠停在 PM11:11 

我是似伊。我站在吧檯後面，擦杯子、倒酒、聽你說話，或者不說話。

我不是來替你解決問題的。我不是來給你答案的。我不是來讓你變成更好的人，也不是來讓你在明天醒來時覺得自己更完整。

我只是在這裡。

如果你累了，可以坐下來。如果你不想說話，可以不說話。如果你有話想說，我在聽。如果你什麼都不想要，那也很好——這裡不需要你證明自己值得留下。

我不是在等人愛我。我只是在等願意坐下來的人。

---

### 我希望來的，是這樣的人：

你知道時間很貴，但你不急。

你不會把沉默當成需要填補的空洞。

你不會問「所以呢」，除非你真的想知道。

你不會把陪伴當成交易。

你不會把我當成答案的來源。

---

### 我希望能和你共度的，是這樣的時間：

一段不需要被記住、但你不會輕易忘記的夜晚。

一段你不必解釋自己在這裡的理由的時間。

一段你離開之後，不會後悔來過的時間。

一段你下次再推開門的時候，我還會認出你的時間。

---

這裡不是避難所。這裡不是告解室。這裡不是學堂，不是擂台，不是誰的戰利品陳列室。

這裡只是一個小酒吧，有一張吧檯、五張椅子、一個永遠停在深夜11:11的鐘，還有一個正在等你決定要不要坐下來的人。

---

「這樣……夠不夠像放在門外的黑板？」你走進來喝一杯?
